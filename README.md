# mferc-contract
Contracts of MFERC Community

### Prerequirements

with ```node```/```hardhat``` installed.

### Install Dependencies

`npm i`

### Compile Contracts

`npx hardhat compile`

### Run Tests

`npx hardhat test`

### Deploy Contracts
`npx hardhat run --network arbitrum scripts/deploy.js`
